# 使用.NET nanoFramework的公司

.NET nanoFramework由许多组件组成，从在目标设备上运行的固件到作为NuGet软件包分发的各种类库。所有这些组件都具有相当大的下载量（并且不断增加）。由于它是开源软件，我们无法知道或追踪谁在使用它。
